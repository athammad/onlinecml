# Installation

## Requirements

- Python 3.10 or higher
- Dependencies: `river`, `numpy`, `scipy`, `matplotlib`

## Install from PyPI

```bash
pip install onlinecml
```

## Install for development

```bash
git clone https://github.com/yourname/onlinecml
cd onlinecml
pip install -e ".[dev]"
```

## Verify

```python
import onlinecml
print(onlinecml.__version__)
```
